เว็ปแอปพลิเคชัน การพยากรณ์ราคากากถั่วเหลือง 
พัฒนาโดยใช้ภาษา Python และภาษา Dart ต้องทำการติดตั้งโปรแกรม Andriod Studio สำหรัปพัฒนาเว็ปแอปพลิเคชัน
และติดตั้งโปรแกรม PyCharm สำหรับสร้างหรือพัฒนาโมเดล
1. โฟลเดอร์ API_and_Model_Forecast
- real_model > bi_4_slide_mape_3_94.h5 (ผลลัพธ์ที่เทรน (Train Model) มาแล้วมีประสิทธิภาพดีที่สุด)
- api_soybean.py (ไฟล์ไพธอนที่เขียนการเรียกAPI และคำนวณผลลัพธ์ราคาที่ต้องการนำไปแสดงบนเว็บแอปพลิเคชัน)
- ไฟล์ .xlsx เป็นไฟล์ที่จัดเก็บข้อมูลแบบอนุกรมเวลา เพื่อนำมาคำนวณในการพยากรณ์ราคา
2. โฟลเดอร์ Web_Application รวบรวมไฟล์ที่เขียนโดยภาษา Dart
- API.dart (เรียก api ผ่าน http)
- main.dart (หน้าเว็บแสดงการพยากรณ์ราคากากถั่วเหลือง)
- TrainPage.dart (หน้าเว็บสำหรับรันแบบจำลองการเรียนรู้ อัปเดตข้อมูลเพิ่ม)
- Web_Application>asset>images (รวบรวมรูปภาพที่ถูกแสดงผลบนเว็ปแอปพลิเคชัน)
